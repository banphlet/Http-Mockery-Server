## Simple http mockey server build with nodejs and nextjs.

[![Greenkeeper badge](https://badges.greenkeeper.io/banphlet/Http-Mockery-Server.svg)](https://greenkeeper.io/)


###  Start dev server 
```sh
> npm run dev

```


### Required envs

|   FIELD                   |          REQUIRED          |
| ------------------------- | -------------------------  |
|          DB_URL           |         YES                |
|   PORT                    |          YES               |
|  JWT_TOKEN                |           YES              |
